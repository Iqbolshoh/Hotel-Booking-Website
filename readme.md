# 🏨 Hotel Booking Website

A modern, responsive and user-friendly **Hotel Booking Landing Page** built using HTML, CSS, and JavaScript.  
This project is designed to showcase hotel rooms, allow users to explore options, and simulate a booking experience with a clean UI/UX design.

## 🎯 Project Purpose

The goal of this project is to build a modern hotel booking landing page that provides users with a smooth and visually appealing experience for exploring hotels and rooms.

Focus:
- Clean UI/UX design
- Responsive layout
- Simple booking interaction
- Real-world landing page structure

---

## ✨ Features

- 🏨 Modern Hotel Landing Page Design
- 🛏️ Rooms Showcase Section
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- ⚡ Fast Loading & Lightweight
- 🎨 Clean and Professional UI
- 📩 Contact Form Section
- 🧾 Booking Modal Popup
- ✨ Smooth Hover Animations
- 🌐 Cross-browser compatible

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla JS)
- Responsive Design Principles
- Mobile First Approach
